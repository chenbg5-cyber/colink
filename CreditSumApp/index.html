📱 חיבור SMS אוטומטי עם Tasker

ברגע שתגדירי את זה - כל SMS מהאשראי יתווסף אוטומטית!

שלב 1: התקנת Tasker

הורידי את Tasker מ-Google Play (עולה כ-15 ש"ח, שווה כל שקל)

שלב 2: יצירת Profile חדש

1. פתחי Tasker → לחצי + → Event

2. בחרי Phone → Received Text

3. בשדה Sender הכניסי את מספרי חברות האשראי שלך (למשל: ויזה, מסטרכרד, ישראכרט). אפשר להוסיף כמה מספרים

שלב 3: יצירת Task

1. לחצי New Task → תני שם "אשראי"

2. לחצי + → Net → Browse URL

3. בשדה URL הדביקי:

https://credit-sum-app.web.app/index.html?shared_text=%smsrb

💡 %smsrb זה משתנה של Tasker שמכיל את תוכן ה-SMS. לחצי על השורה להעתקה.

שלב 4: בדיקה

שלחי לעצמך SMS עם טקסט כזה:
עסקה בכרטיס *1234 בסך 50.00 ש״ח אושרה
האפליקציה אמורה להוסיף ₪50 אוטומטית!
